# Ocean Blue – Luxury Zanzibar Experiences

Ocean Blue is a **React-based luxury travel website** that showcases curated ocean excursions and private island transfers in Zanzibar. This project serves as a practice playground for React, Tailwind CSS, and front-end UI development, simulating a high-end booking experience without a back-end.

## Features

- **Responsive UI**: Designed with Tailwind CSS for a clean, modern, and mobile-friendly interface.
- **Home Page Sections**: Hero, About, Experiences, Private Transfers, Journal, and Contact.
- **Experiences Page**: Large horizontal cards highlighting curated excursions with a “View Photos” button.
- **Gallery Page**: Sections displaying images for each experience in a vertical grid layout.
- **Booking Simulation**: Interactive booking engine that calculates total cost based on:
  - Selected excursion
  - Number of adults & children (children pay half)
  - Optional lunch add-ons
  - Private transfers
- **Contact Form**: Name, Email, Enquiry fields with validation using `react-hook-form`.
- **Smooth Scroll and Navigation**: Scroll-to-top button and anchor links with smooth behavior.
- **Reusable Components**: Navbar, Footer, Card layouts, and Buttons styled according to the brand.

## Tech Stack

- **React** – Front-end framework  
- **React Router** – Routing and page navigation  
- **React Hook Form** – Form handling and validation  
- **Tailwind CSS** – Styling and responsive design  
- **React Icons** – Iconography for social links and UI elements
